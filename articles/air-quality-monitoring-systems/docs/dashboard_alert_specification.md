# Dashboard and Alert Specification

Public communication products should distinguish pollutant-specific concentration records from index values, alerts, forecasts, exposure estimates, and public-health guidance.

## Required fields

- pollutant and averaging period
- source dataset
- platform type
- data freshness
- calibration status
- QC status
- public-health category or threshold
- affected geography
- vulnerable-population caveat
- uncertainty statement
- action guidance
- last updated timestamp
- responsible agency or owner
