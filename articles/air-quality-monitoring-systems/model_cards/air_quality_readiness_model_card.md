# Air Quality Monitoring Readiness Model Card

## Intended use

- evaluate calibration quality
- evaluate data completeness
- evaluate spatial coverage
- evaluate temporal coverage
- evaluate meteorological context
- evaluate exposure relevance
- evaluate uncertainty communication
- evaluate governance readiness
- triage high-stakes air-quality claims

## Not intended for

- replacing regulatory methods
- replacing epidemiological exposure assessment
- certifying source attribution
- using low-cost sensor values without caveats
- hiding uncertainty, missingness, or siting limitations
