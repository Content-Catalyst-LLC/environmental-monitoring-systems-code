# Land Monitoring Evidence Framework

Land-use monitoring and environmental change detection are infrastructures of temporal landscape evidence.

## Core layers

| Layer | Question | Evidence artifact |
|---|---|---|
| Objective | What geography, land process, temporal window, class system, and decision use are being monitored? | Monitoring objective manifest |
| Observation | Which imagery, sensors, dates, masks, and field records support the analysis? | Imagery inventory and observation metadata |
| Preprocessing | How are observations corrected, masked, composited, and harmonized through time? | Preprocessing manifest |
| Classification | How are land-cover, land-use, condition, or disturbance classes assigned? | Classification manifest and model card |
| Change logic | Which transitions, thresholds, persistence rules, and recovery signals count as change? | Change-rule registry |
| Validation | Which reference samples test classification and change claims? | Reference sample registry and accuracy report |
| Uncertainty | How are confidence, class error, change error, and area-estimation uncertainty communicated? | Validation report and uncertainty statement |
| Indicator | How are land-change outputs translated into carbon, biodiversity, watershed, degradation, restoration, or planning metrics? | Indicator specification |
| Governance | Who owns revisions, public caveats, dispute review, and reporting decisions? | Governance log |
