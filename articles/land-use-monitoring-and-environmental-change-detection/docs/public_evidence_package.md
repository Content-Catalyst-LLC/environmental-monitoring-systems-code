# Public Evidence Package Template

A public evidence package for land-use monitoring should explain what a map product can and cannot support.

## Required public-facing elements

- monitored geography
- monitoring period
- imagery sources
- land-class legend
- distinction between land cover, land use, land condition, and land change
- classification method
- change-detection method
- validation sample source
- overall accuracy
- class-specific accuracy where available
- change confidence
- uncertainty and caveats
- known blind spots
- revision history
- governance owner
- valid-use statement

## Core warning

A land-cover class should not be treated as a complete description of land use, ecological condition, ownership, tenure, or governance status.
