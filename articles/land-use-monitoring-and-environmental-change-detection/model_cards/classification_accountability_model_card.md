# Classification Accountability Model Card

## Purpose

This placeholder card documents a future classification or change-detection model.

## Intended use

- land-cover classification
- land-use interpretation support
- condition-change detection
- disturbance and recovery detection
- transition confidence scoring
- public evidence review

## Required documentation

- model ID
- model version
- class legend version
- training sample source
- validation sample source
- imagery source
- seasonal window
- geography of intended use
- known class confusions
- validation status
- uncertainty statement
- valid-use limits

## Known risks

- conflating land cover and land use
- hiding within-class degradation
- treating seasonal variation as durable change
- class imbalance
- biased reference data
- overclaiming precision in public reporting
