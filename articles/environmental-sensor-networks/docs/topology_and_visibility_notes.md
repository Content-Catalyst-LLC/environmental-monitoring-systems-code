# Topology and Visibility Notes

Environmental sensor networks are selective visibility systems.

## Topology questions

- What environmental process is the topology designed to observe?
- Are gradients, hotspots, reference areas, vulnerable communities, vertical layers, and boundary zones represented?
- Which areas are intentionally unmonitored?
- Which areas are unintentionally unmonitored?
- Where does the network create false reassurance through sparse coverage?
- Which nodes provide redundancy for high-stakes decisions?

## Visibility questions

- Which variables are directly measured?
- Which variables are proxies?
- Which environmental states remain inferred?
- Which communities, habitats, tributaries, corridors, or risk zones remain under-observed?
- Does the network support public contestability and independent review?

## Action questions

- What decisions depend on the network?
- What latency is required?
- Who owns alerts?
- What happens after a threshold is crossed?
- How are false positives and false negatives reviewed?
