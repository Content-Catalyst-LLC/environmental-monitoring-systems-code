# Risk and Resilience Monitoring Framework

Environmental risk and resilience should be monitored as dynamic system conditions.

## Core dimensions

| Dimension | Question | Evidence artifact |
|---|---|---|
| Hazard | What environmental disturbance is possible or occurring? | Hazard inventory, event catalog |
| Exposure | Who and what are in harm's way? | Exposure registry, GIS layer |
| Vulnerability | Which conditions increase susceptibility to harm? | Vulnerability profile |
| Capacity | Which resources reduce harm or support response? | Capacity profile |
| Threshold | Which limits or triggers are being approached? | Threshold registry |
| Impact | What damage or disruption occurred? | Impact assessment |
| Recovery | Are essential functions returning over time? | Recovery trajectory table |
| Adaptive capacity | Is the system learning and reducing future risk? | Adaptive-capacity assessment |
| Equity | Is risk, warning, response, and recovery unevenly distributed? | Equity and coverage audit |
