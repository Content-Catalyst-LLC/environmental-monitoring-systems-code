# Public Evidence Package Template

A public evidence package for biodiversity monitoring should explain what a biodiversity claim can and cannot support.

## Required public-facing elements

- monitoring geography
- monitoring period
- target taxa and biodiversity dimensions
- observation methods
- effort and sampling design
- detectability treatment
- taxonomic and spatial coverage
- indicator definitions and EBV mapping
- habitat and ecosystem context
- community knowledge governance notes where applicable
- uncertainty statement
- known blind spots
- valid-use statement
- governance owner
- revision history

## Core warning

A species count, richness metric, habitat map, or biodiversity indicator should not be treated as a complete description of biodiversity condition, population viability, ecological integrity, or conservation success.
