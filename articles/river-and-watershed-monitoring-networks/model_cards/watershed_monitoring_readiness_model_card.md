# Watershed Monitoring Readiness Model Card

## Intended use

- evaluate gauge coverage
- evaluate water-quality coverage
- evaluate ecological coverage
- evaluate network representativeness
- evaluate temporal continuity
- evaluate event capture
- evaluate uncertainty communication
- evaluate interoperability
- evaluate governance readiness
- triage high-stakes watershed claims

## Not intended for

- replacing hydrologist or ecologist judgment
- certifying basin health
- replacing official water-quality assessments
- treating station count as evidence quality
- hiding blind spots, uncertainty, or governance limitations
