# Watershed Monitoring Evidence Framework

| Layer | Question | Evidence artifact |
|---|---|---|
| Objective | Which basin, decision use, target variables, agencies, and time horizon are being supported? | Watershed objective manifest |
| Topology | How do headwaters, tributaries, confluences, main stems, outlets, reservoirs, and groundwater-connected reaches relate? | Basin topology graph |
| Nodes | Which sites observe flow, quality, ecology, meteorology, groundwater, or infrastructure operations? | Monitoring node registry |
| Drivers | Which hydrological and human drivers shape basin response? | Hydrological driver matrix |
| Telemetry | Are records timely, complete, quality-flagged, and station-health aware? | Telemetry and station-health log |
| Load | Are concentration and discharge paired well enough to estimate mass transport? | Water-quality and load records |
| Ecology | Are habitat and biological responses comparable across sites and time? | Ecological assessment registry |
| Representativeness | Which headwaters, tributaries, events, groundwater pathways, or jurisdictions remain weakly observed? | Representativeness audit |
| Governance | How are data shared, caveated, revised, and used across agencies? | Governance and data-sharing log |
