# Representativeness Audit Template

A watershed monitoring network should periodically review whether its stations still represent the basin questions being asked.

## Review questions

1. Which headwaters and tributaries remain ungauged?
2. Which sub-basins have water-quality stations but no paired flow records?
3. Which ecological sites lack hydrological or chemical driver context?
4. Which monitoring records are too coarse for storm, flood, or first-flush events?
5. Which groundwater-connected reaches remain weakly observed?
6. Which urbanizing, agricultural, industrial, or restoration zones are underrepresented?
7. Which transboundary areas have data discontinuities or method differences?
8. Which legacy stations remain valuable for long-term continuity?
9. Which stations no longer match emerging climate, land-use, or infrastructure pressures?
10. Which public claims exceed what the network can support?
