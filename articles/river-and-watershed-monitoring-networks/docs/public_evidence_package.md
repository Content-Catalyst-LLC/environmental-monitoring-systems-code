# Public Evidence Package Template

Required public-facing fields:

- watershed name and basin boundary
- monitoring purpose and decision use
- monitoring period
- station list and site types
- network topology summary
- measured variables
- telemetry and station-health caveats
- water-quality and load methods
- ecological assessment protocols
- representativeness limitations
- event-capture limitations
- data-sharing and interoperability notes
- uncertainty statement
- valid-use statement
- governance owner
- revision history
