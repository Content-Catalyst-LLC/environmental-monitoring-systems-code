# Coverage and Blind-Spot Audit Template

A weather sensing network should periodically review whether its observations remain adequate for the operational decisions being made.

## Review questions

1. Which forecast or warning decisions depend on this observing network?
2. Which platforms are active, degraded, stale, or missing?
3. Which geographic regions remain weakly observed?
4. Which vertical atmospheric structures are undersampled?
5. Which marine, mountain, rural, polar, or lower-capacity areas remain weakly covered?
6. Which radar beam, range, or blockage gaps affect hazard detection?
7. Which satellite products are indirect retrievals requiring uncertainty caveats?
8. Which observations arrive too late for nowcasting or warning use?
9. Which valid observations are still not assimilable because of metadata or format limitations?
10. Which public claims exceed the observation system's evidence strength?
