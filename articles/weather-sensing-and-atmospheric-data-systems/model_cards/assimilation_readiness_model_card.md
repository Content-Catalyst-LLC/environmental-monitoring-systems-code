# Assimilation Readiness Model Card

## Intended use

- evaluate whether observations are eligible for model or nowcast ingestion
- track metadata completeness
- track QC status
- track latency within operational budget
- track supported format status
- track error-model or uncertainty readiness

## Not intended for

- replacing operational meteorologist judgment
- certifying forecast skill
- treating all valid observations as assimilable
- hiding platform bias or coverage limitations
- using stale observations in time-sensitive decisions
