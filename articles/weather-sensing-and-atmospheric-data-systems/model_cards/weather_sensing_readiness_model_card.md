# Weather Sensing Readiness Model Card

## Intended use

- evaluate observation freshness
- evaluate spatial coverage
- evaluate vertical coverage
- evaluate QC completeness
- evaluate assimilation readiness
- evaluate representativeness
- evaluate uncertainty communication
- evaluate governance readiness
- triage high-stakes operational weather claims

## Not intended for

- replacing official forecast verification
- certifying observational completeness
- treating observation count as evidence quality
- hiding observational inequality
- masking stale or non-assimilable observations
