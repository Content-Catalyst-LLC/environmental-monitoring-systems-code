# Soil Monitoring Evidence Framework

| Layer | Question | Evidence artifact |
|---|---|---|
| Objective | What crop, field, soil type, management zone, temporal horizon, and decision use are being monitored? | Monitoring objective manifest |
| Soil context | What texture, depth, drainage, rooting zone, organic matter, and management history shape interpretation? | Soil profile registry |
| Observation | Which sensors, samples, lab tests, and remote products support the claim? | Sensor registry, lab results, remote-product linkage |
| Depth | What part of the profile is represented? | Depth plan and root-zone interpretation |
| Calibration | Are sensors valid for local soil and installation conditions? | Calibration and maintenance log |
| Telemetry | Are records current, complete, and quality-flagged? | Telemetry completeness report |
| Decision | Which irrigation, nutrient, salinity, drought, runoff, or stewardship decision is supported? | Threshold and decision registry |
| Governance | Who owns revisions, public caveats, and reporting responsibilities? | Governance log |
