# Ecological Indicator Model Card

## Purpose

This placeholder card documents a future ecological indicator or assessment model.

## Intended use

- condition assessment
- structure monitoring
- function monitoring
- integrity screening
- recovery trajectory review
- resilience screening
- public evidence review

## Required documentation

- indicator ID
- ecological dimension
- proxy meaning
- baseline or reference condition
- threshold or interpretation rule
- input observations
- spatial and temporal scale
- validation status
- uncertainty statement
- valid-use limits
- governance owner

## Known risks

- treating proxies as ecosystems
- conflating extent and condition
- hiding poor function evidence inside structural metrics
- overstating resilience from short-term stability
- underrepresenting inaccessible or marginalized ecosystems
