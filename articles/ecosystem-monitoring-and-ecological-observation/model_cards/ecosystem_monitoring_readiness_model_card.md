# Ecosystem Monitoring Readiness Model Card

## Model purpose

This placeholder model card documents an ecosystem-monitoring readiness and ecological evidence-quality scoring workflow.

## Intended use

- evaluate extent visibility
- evaluate condition evidence
- evaluate structure evidence
- evaluate function evidence
- evaluate integrity evidence
- evaluate resilience evidence
- evaluate baseline strength
- evaluate uncertainty communication
- evaluate governance readiness
- triage high-stakes ecological claims

## Not intended for

- replacing ecological expertise
- replacing field validation
- treating a composite score as ecosystem truth
- certifying restoration or resilience claims without trajectory evidence
- hiding weak ecological dimensions inside a final rating

## Inputs

- extent visibility
- condition evidence
- structure evidence
- function evidence
- integrity evidence
- resilience evidence
- baseline strength
- uncertainty communication
- governance readiness
- high-stakes use flag

## Known limitations

- simplified scoring
- ecological dimensions are context dependent
- resilience requires longer time series than many programs have
- integrity is a synthetic judgment, not a directly observed variable
- public accountability requires evidence beyond a score
