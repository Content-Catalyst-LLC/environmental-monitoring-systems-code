# Field Observation Protocol

This protocol scaffold supports ecosystem monitoring programs that combine field observation, sensor records, remote sensing, and ecological indicators.

## Minimum field-record fields

- site ID
- ecosystem type
- geography
- latitude and longitude
- observation date
- observer or method
- protocol version
- observed variable
- unit
- ecological dimension
- indicator linkage
- quality flag
- uncertainty note
- baseline or reference relation
- disturbance context
- governance owner

## Protocol principles

1. Separate observation from assessment.
2. Do not treat extent as condition.
3. Document whether each indicator represents structure, function, condition, integrity, resilience, disturbance, or recovery.
4. Preserve method version and observer/protocol changes.
5. Record disturbance and management context.
6. Report uncertainty and proxy limits.
7. Maintain public evidence notes for high-stakes ecological claims.
