# Public Evidence Package Template

A public evidence package for ecosystem monitoring should explain what an ecosystem assessment can and cannot support.

## Required public-facing elements

- ecosystem type and geography
- monitoring period
- field and remote-sensing methods
- indicator definitions
- ecological dimensions represented
- baseline or reference condition
- disturbance and recovery context
- validation summary
- uncertainty statement
- proxy limits
- representativeness caveats
- governance owner
- valid-use statement
- revision history

## Core warning

A mapped ecosystem or single ecological indicator should not be treated as a complete description of ecosystem condition, function, integrity, or resilience.
