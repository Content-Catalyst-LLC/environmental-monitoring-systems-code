# Mathematical Lens

Ecosystem monitoring should be evaluated as a system of ecological evidence, not only as a collection of observations or indicators.

## Ecological condition synthesis

\[
C_{\mathrm{condition}} = \frac{\sum_{i=1}^{n} w_i I_i}{\sum_{i=1}^{n} w_i}
\]

Ecosystem condition can be represented as a weighted synthesis of indicators, but the result depends on indicator choice, weighting, baseline, and ecological interpretation.

## Disturbance regime

\[
D_{\mathrm{disturbance}} = f(F_{\mathrm{frequency}}, I_{\mathrm{intensity}}, A_{\mathrm{extent}}, T_{\mathrm{duration}})
\]

Disturbance regime depends on frequency, intensity, spatial extent, and duration rather than on occurrence alone.

## Recovery ratio

\[
R_{\mathrm{recovery}} = \frac{C_{t+k} - C_t}{C_{\mathrm{reference}} - C_t}
\]

Recovery ratio compares observed condition improvement with the gap between disturbed condition and reference condition.

## Integrity synthesis

\[
I_{\mathrm{integrity}} = g(S_{\mathrm{structure}}, F_{\mathrm{function}}, B_{\mathrm{biotic}}, P_{\mathrm{process}}, C_{\mathrm{connectivity}})
\]

Ecosystem integrity is a synthetic judgment about structure, function, biotic composition, ecological processes, and connectivity.

## Ecosystem evidence quality

\[
Q_{\mathrm{ecosystem\ evidence}} = w_1E_x + w_2C_d + w_3S_t + w_4F_n + w_5I_g + w_6R_s + w_7B_r + w_8U_c + w_9G_r
\]

Ecosystem evidence quality depends on extent, condition, structure, function, integrity, resilience, baseline strength, uncertainty communication, and governance readiness.
