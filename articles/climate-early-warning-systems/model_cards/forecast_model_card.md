# Forecast Model Card

## Intended use

- support warning decisions
- estimate hazard timing and intensity
- provide forecast confidence and valid horizon
- document uncertainty and limitations
- support impact-based warning interpretation

## Required documentation

- forecast source
- model or expert-analysis source
- issue time
- valid start and end time
- forecast horizon
- forecast confidence
- expected impact time
- uncertainty statement
- known failure modes
- valid-use limits

## Core caveat

A forecast is not a warning. A forecast becomes part of an early warning only when it is connected to risk knowledge, communication, preparedness, and action.
