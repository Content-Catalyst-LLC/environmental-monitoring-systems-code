# Public Evidence Package Template

Required public-facing fields:

- hazard type and geography
- warning period
- warning authority
- protected population
- forecast source and valid horizon
- forecast confidence and uncertainty statement
- risk-knowledge source
- exposure and vulnerability caveats
- impact-threshold source
- protective lead-time statement
- communication channels
- action guidance
- inclusion and accessibility caveat
- false-alarm / missed-event review status
- governance owner
- revision history
