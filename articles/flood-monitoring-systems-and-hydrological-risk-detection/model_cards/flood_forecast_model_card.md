# Flood Forecast Model Card

## Required documentation

- forecast source
- basin or forecast point
- model type
- input observations
- rainfall source
- soil-wetness or snowmelt context
- reservoir or operations context
- forecast horizon
- forecast update frequency
- calibration and validation summary
- uncertainty statement
- known failure modes
- valid-use limits
- warning authority and response owner

## Core caveat

A flood forecast is not a guarantee of local consequence. It must be translated through thresholds, inundation maps, exposure, vulnerability, communication, and response capacity.
