# Post-Event Review Template

A flood monitoring system should learn from events, near misses, false alarms, and missed warnings.

## Review questions

1. Were gauges and rainfall products operational?
2. Were telemetry records complete and timely?
3. Did rate-of-rise alerts trigger correctly?
4. Did forecast timing and crest estimates match observed outcomes?
5. Did inundation maps match observed flooded areas?
6. Were thresholds and impact descriptions accurate?
7. Were warnings issued early enough?
8. Did public communication reach affected communities?
9. Were road closures, evacuation support, and infrastructure operations timely?
10. Which blind spots, vulnerable populations, or ungauged areas need priority upgrades?
