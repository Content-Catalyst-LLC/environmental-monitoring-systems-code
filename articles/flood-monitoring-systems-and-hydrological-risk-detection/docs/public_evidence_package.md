# Public Evidence Package Template

Required public-facing fields:

- flood type and geography
- monitoring period
- monitored gauges and rainfall sources
- telemetry and latency caveats
- forecast model or guidance product
- flood thresholds and local impacts
- inundation-map source and assumptions
- uncertainty statement
- lead-time statement
- warning channels
- response pathway
- known blind spots
- vulnerable-area caveats
- governance owner
- revision history
