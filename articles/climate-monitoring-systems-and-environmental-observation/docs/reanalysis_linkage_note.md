# Reanalysis Linkage Note

Reanalysis products are model-assisted reconstructions, not direct observations. Climate monitoring systems should document:

- observation sources assimilated or used for validation
- model or reanalysis product version
- temporal coverage
- spatial resolution
- variables
- known caveats
- uncertainty and ensemble structure where available
- differences between direct observation, gridded record, and reanalysis
- citation and access path
