# Public Evidence Package Template

Required public-facing fields:

- waterbody type and location
- monitoring purpose and decision use
- designated use or exposure pathway
- monitoring period
- parameter list and method summary
- sensor and laboratory methods
- QA/QC status
- calibration and maintenance status
- sampling design and representativeness caveat
- hydrological context and event status
- threshold or standard source
- uncertainty statement
- valid-use statement
- capacity or coverage caveat
- governance owner
- revision history
