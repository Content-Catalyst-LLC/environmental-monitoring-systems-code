# Water Quality Monitoring Evidence Framework

| Layer | Question | Evidence artifact |
|---|---|---|
| Objective | Which waterbody, use, risk question, decision context, and parameter rationale are being supported? | Water quality objective manifest |
| Site | Which location, depth, waterbody type, watershed position, and exposure context does the record represent? | Site and station registry |
| Method | Which field, sensor, sample, laboratory, microbial, or biological method produced the observation? | Parameter and method registry |
| QA/QC | Are calibration, fouling, blanks, duplicates, detection limits, and review flags documented? | QA/QC policy and calibration log |
| Sample | How was the sample collected, preserved, transported, and analyzed? | Sampling event log and chain-of-custody record |
| Hydrology | What flow, stage, event, runoff, dilution, or load context applies? | Hydrology and load linkage |
| Assessment | Which standards, criteria, advisories, operational thresholds, or ecological benchmarks apply? | Assessment threshold registry |
| Governance | Can public claims be inspected, caveated, revised, and tied to responsible owners? | Governance and public evidence log |
