# Load and Threshold Model Card

## Required documentation

- site ID
- parameter ID
- concentration source
- discharge or hydrological source
- event status
- load calculation method
- threshold source
- designated or decision use
- unit consistency
- uncertainty statement
- valid-use limits

## Core caveat

Concentration, load, exceedance, and exposure are distinct evidence types. A monitoring claim should state which one is being used and why.
