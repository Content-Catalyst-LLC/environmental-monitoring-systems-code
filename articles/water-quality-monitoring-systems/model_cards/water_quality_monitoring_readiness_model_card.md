# Water Quality Monitoring Readiness Model Card

## Intended use

- evaluate sampling representativeness
- evaluate analytical validity
- evaluate hydrological context
- evaluate QA/QC completeness
- evaluate load readiness
- evaluate threshold alignment
- evaluate uncertainty communication
- evaluate data interoperability
- evaluate governance readiness
- triage high-stakes water-quality claims

## Not intended for

- replacing official regulatory assessment
- replacing public-health judgment
- certifying drinking-water safety
- replacing laboratory confirmation
- treating proxy signals as definitive contaminant evidence
- hiding uncertainty, capacity limits, or representativeness gaps
