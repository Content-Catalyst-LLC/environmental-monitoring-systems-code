# TinyML Edge Inference Evidence Placeholder

This directory is reserved for a future TinyML example that produces local evidence to complement satellite products.

Possible use cases:

- local flood confirmation
- smoke or air-quality anomaly detection
- field vegetation stress proxy
- snow or ice condition confirmation
- cloud-edge quality flagging for field cameras

The TinyML workflow should preserve:

- station ID
- linked satellite product ID
- model ID
- model version
- inference confidence
- observed versus inferred distinction
- uncertainty label
- timestamp
- evidence-retention policy
